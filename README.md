# 🛒 Grocery E-Commerce Website (MERN Stack)

## 📌 Project Description

This project is a **Grocery E-Commerce Website** developed using the **MERN Stack** (MongoDB, Express.js, React.js, Node.js).
It allows users to browse grocery products, add items to the cart, and place orders online.
The project also includes a **Seller/Admin dashboard** to manage products and orders.

---

## 🚀 Features

* User Registration & Login
* Browse Grocery Products
* Product Categories
* Add to Cart
* Checkout System
* Order Management
* Seller/Admin Dashboard
* Add, Update and Delete Products

---

## 🛠️ Technologies Used

* **Frontend:** React.js, Vite, CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Version Control:** Git & GitHub

## ⚙️ How to Run the Project

### 1️⃣ Clone the repository

```
git clone https://github.com/ananyabelagal/Grocery-Ecommerce-Website.git
```

### 2️⃣ Install dependencies

Frontend:

```
cd client
npm install
npm run dev
```

Backend:

```
cd backend
npm install
npm run dev
```
